# Scholarly Library

A full-stack library management system built with React and Node.js.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB
- **Auth:** JWT with email verification

## Features

- User and Admin roles
- Book/material catalog with search and filter
- Borrow request management
- Save history and activity feed
- Dark premium UI with gold accents

## Getting Started

### Backend
```bash
cd backend
cp .env.example .env   # fill in your values
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
